# Zanzavat NGO Image Management System

This folder structure contains all the static assets for the Zanzavat Bahuudeshiya Shaikshanik Sanstha website. To update images on the site, simply upload the new images into their corresponding directories using the naming and sizing guidelines below.


---

## Directory Reference & Sizing Guide

| Folder Path | Description / Use Case | Recommended Sizing |
| :--- | :--- | :--- |
| `/logo` | NGO Logos (Navbar, Footer, Favicons) | `512x512px` (PNG, transparent background) |
| `/hero` | Full-screen background carousel images for Home page hero section | `1920x1080px` (WEBP or compressed JPG) |
| `/about` | Photos for history, mission, vision, and timeline sections | `800x600px` (Horizontal or Square) |
| `/impact` | Photo-driven stories of beneficiaries and community reach | `1000x660px` |
| `/education-support` | Images representing student sponsorships, books, and study aids | `800x600px` |
| `/medical-camps` | Photos from free healthcare, eye-checkups, and medicine distribution camps | `800x600px` |
| `/food-distribution` | Food drives, grocery kits, and meals distribution photos | `800x600px` |
| `/clothing-drives` | Distribution of warm clothes, blankets, and school uniforms | `800x600px` |
| `/student-welfare` | General student and child welfare programs and workshops | `800x600px` |
| `/volunteers` | Photos of volunteers, group sessions, and community events | `800x600px` |
| `/leadership` | Executive profile pictures (Ajinkya Bhakre, Arya Sontake) | `600x600px` (Square profile format) |
| `/gallery` | Masonry layout pictures and spotlight scroll images | `1200x800px` (High resolution) |
| `/events` | Specific folder for covers and inline details of various events | `1000x600px` |
| `/events/event-template` | Reference images for custom, newly-added events | `1000x600px` |
| `/donations` | Images for QR code placeholders, bank transfer guides, and donation story banners | `600x600px` (QR), `1000x600px` (Banner) |
| `/testimonials` | Quotes and profiles of beneficiaries or partners | `300x300px` (Square profile) |

---

## Best Practices
1. **Format**: Use **WEBP** format where possible to ensure fast load speeds. If WEBP is unavailable, use highly-compressed **JPG**. Use transparent **PNG** only for logos and overlays.
2. **Compression**: Always compress images using tools like TinyPNG/Squoosh before uploading. Keep file sizes under `200KB` for regular content and `450KB` for full-screen hero backgrounds.
3. **Naming**: Use hyphen-separated lowercase names (e.g. `education-camp-1.webp` instead of `Education Camp 1 (New) final.JPG`).
